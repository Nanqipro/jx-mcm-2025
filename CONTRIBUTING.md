# 参与贡献

欢迎通过 Issue 或 Pull Request 改进数据解析、模型实现、可视化与复现实验。

提交前请确认：

1. 不上传竞赛原始数据、机场视频、行业标准全文或其他无再发布授权的材料；
2. 新增实验说明数据来源、随机种子、训练/测试划分和评价指标；
3. 运行 `python -m unittest discover -s tests -v` 与 `python -m compileall -q src scripts tests`；
4. 不把竞赛实验结果描述为可直接用于机场运行决策的生产系统。

如修复模型问题，请在 Pull Request 中说明影响的题目、输入字段和结果变化。
