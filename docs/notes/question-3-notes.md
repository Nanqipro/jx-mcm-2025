# **基于大雾背景视频学习的能见度回归建模**

## **问题三:能见度与有雾视频图像回归关系模型建立**

### **1\. 问题背景与理解**

#### **1.1 问题描述**

建立能见度与有雾视频图像之间回归关系模型,并对回归的精度进行估计。同时,考虑到雾的大小与近地面的气象要素(温度、湿度、风速等)密切相关,为了提高能见度估计的精度,需要在回归分析时添加适当的气象要素。

#### **1.2 问题理解**

这是一个典型的多元回归建模问题,需要:

* 建立能见度(因变量)与图像特征、气象要素(自变量)的数学关系  
* 给出明确的数学公式,而非黑盒机器学习模型  
* 评估模型的预测精度和实用价值  
* 解释模型的物理意义和实际应用价值

#### **1.3 理论基础**

根据题目提供的物理公式:

"F=F0​⋅e∧(−σz)" (光强衰减方程)

"MOR=3.912/σ" (气象光学视程)

其中 σ 为衰减系数,与雾的厚度相关。这为我们的建模提供了物理理论支撑。

### **2\. 数据探索与分析**

#### **2.1 数据基本信息**

* 数据规模:2774个样本,29个特征  
* 数据质量:无缺失值,数据完整性良好  
* 时间跨度:2020年3月12日的连续监测数据  
* 采样间隔:15秒间隔的视频帧数据

#### **2.2 目标变量选择分析**

数据包含4个能见度相关指标:

* visibility\_vis\_1a: 均值3043.5米  
* visibility\_vis\_10a: 均值3568.6米  
* visibility\_mor\_raw: 均值2772.0米(气象光学视程)  
* visibility\_vis\_raw: 均值3817.1米

选择决策:选择 visibility\_mor\_raw 作为目标变量,理由如下:

1. 标准性:气象光学视程(MOR)是国际标准的能见度度量指标  
2. 物理意义:直接对应题目给出的物理公式MOR \= 3.912/σ  
3. 相关性强:与其他能见度指标高度相关(r\>0.9)但独立性更好  
4. 数据范围:56-13951米,涵盖了从重雾到清晰的完整范围

### **3\. 数据预处理与特征工程**

#### **3.1 多重共线性问题识别**

通过计算方差膨胀因子(VIF),发现严重的多重共线性问题:

| 特征 | VIF值 | 共线性程度 |
| :---- | :---- | :---- |
| local\_var\_mean | 6136.8 | 极严重 |
| laplacian\_var | 2411.9 | 极严重 |
| brenner | 686.8 | 严重 |

发现关键问题:

* laplacian\_var与 local\_var\_mean 完全相关 (r=1.000)  
* 多个图像特征间存在高度相关性 (r\>0.9)  
* 气象要素间也存在高相关性

#### **3.2 特征选择策略**

基于相关性分析和VIF检查,采用手动特征选择策略:

选择原则:

1. 从每组高相关特征中选择一个代表  
2. 保留与目标变量相关性最强的特征  
3. 确保气象和图像特征的平衡

最终选择的特征:

* 图像特征: laplacian\_var (代表清晰度), high\_freq\_ratio (高频信息)  
* 气象特征: weather\_humidity\_pct (湿度), weather\_temperature\_c (温度), wind\_wind\_speed\_10m (风速)

#### **3.3 数据变换**

##### **3.3.1 目标变量变换**

分析目标变量分布特征:

* 原始分布:偏度=2.318,严重右偏  
* 对数变换:偏度=-0.954,过度矫正  
* 平方根变换:偏度=0.485,最接近正态分布

决策:采用平方根变换,使目标变量更符合线性回归的正态性假设。

##### **3.3.2 特征标准化**

由于特征量纲差异巨大(如brenner达到上亿量级),采用Z-score标准化:

Xstd​=(X−μ)/σ

### **4\. 模型建立与比较**

#### **4.1 初步模型选择策略**

1. 普通线性回归:最基础的线性关系  
2. 岭回归(Ridge):处理多重共线性  
3. Lasso回归:自动特征选择  
4. 弹性网络:结合岭回归和Lasso的优点

#### **4.4 模型局限性分析与改进思路**

线性模型的局限性发现:

1. 现实复杂性:能见度与各影响因子的关系往往是非线性的  
2. 交互效应忽略:忽略了图像特征与气象要素之间的相互作用  
3. 二次关系缺失:某些特征可能存在饱和效应或阈值效应

改进建模思路:

基于对雾形成机理的深入分析,我们发现:

* 湿度与温度存在强交互效应(德涅温度效应)  
* 图像清晰度与雾浓度呈非线性关系  
* 高频信息比例可能存在饱和效应

因此,我们提出多项式回归建模方案。

#### **4.5 多项式回归模型建立**

##### **4.5.1 理论基础**

多项式回归能够捕捉:

* 非线性关系:特征的二次效应  
* 交互效应:特征间的相互作用  
* 复杂模式:更接近真实物理过程

模型形式:

visibility=β0​+Σ(αi​Xi​)+Σ(βi​Xi​2)+Σ(γi j​Xi​Xj​)+ϵ

其中:

* 第一项:线性项  
* 第二项:二次项(非线性效应)  
* 第三项:交互项(特征相互作用)

##### **4.5.2 多项式特征构建**

使用Polynomial Features(degree=2)将5个核心特征扩展为20个多项式特征:

特征扩展过程:

* 原始特征:5个  
* 线性项:5个  
* 二次项:5个  
* 交互项:10个  
* 总计:20个多项式特征

3. 实用性更好:  
   * 预测精度接近工程要求

### **7\. 模型解释与物理意义**

#### **7.1 非线性效应分析**

##### **7.1.1 高频信息比例的饱和效应**

发现:高频信息比例存在显著的二次项效应

物理解释:

* 线性系数:+684,267(正效应)  
* 二次系数:-177,320(饱和效应)  
* 当雾较浓时,增加图像处理能显著改善能见度估计  
* 当图像已经很清晰时,继续提升图像质量的边际效应递减  
* 这符合视觉感知的Weber-Fechner定律

##### **7.1.2 温度的非线性特征**

二次项系数:+60.70

物理意义:

* 温度对能见度的影响在高温时更加显著  
* 体现了热力对流对雾消散的加速作用  
* 符合大气边界层理论

#### **7.2 交互效应深度分析**

##### **7.2.1 图像-气象交互机制**

1. 高频比例×温度交互(-16,849):  
   * 机理:温度影响雾滴大小分布,进而影响光散射特性  
   * 应用:高温条件下,图像特征的权重应当调高  
   * 验证:与Mie散射理论一致  
2. 高频比例×湿度交互(-3,646):  
   * 机理:湿度决定雾的生消,影响图像特征的稳定性  
   * 应用:高湿度时图像特征更可靠  
   * 验证:与相对湿度理论吻合  
3. 高频比例×风速交互(-12,399):  
   * 机理:风速影响雾的水平分布均匀性  
   * 应用:大风时图像特征代表性更强  
   * 验证:与湍流输送理论符合

##### **7.2.2 气象要素间交互**

湿度×温度交互(+96.32):

* 气象学意义:体现了德涅温度效应  
* 物理过程:饱和水汽压与温度的指数关系  
* 实际意义:高温高湿时雾更难形成
