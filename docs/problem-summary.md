# 赛题摘要：基于大雾背景视频的能见度回归建模

> 本页是对 2025 年江西省数学建模竞赛参赛题目的概述，不收录题面全文。题面、数据与行业标准的版权归各自权利人所有。

赛题围绕机场低能见度场景，要求利用跑道监控视频和自动气象观测系统（AMOS）数据，研究图像模糊、能见度连续变化以及多源回归估计问题。

## 三个建模任务

1. **图像模糊度量化**：从视频帧提取清晰度、边缘、频域和对比度等特征，建立能够反映雾致模糊程度的数学指标。
2. **连续变化建模**：针对雾与视频随时间连续演化的特性，建立能见度的连续时间或状态空间模型。
3. **多源能见度回归**：融合视频图像特征与温度、湿度、风速、气压等近地面气象要素，估计能见度并评价回归误差。

## 数据角色

- 机场跑道监控视频：用于抽帧和提取图像特征；
- AMOS 能见度观测：提供 RVR、MOR 等监督或验证信号；
- 温湿压与风场观测：作为影响雾形成、维持和消散的外生变量；
- 数据格式说明与行业标准：用于解释字段和原始记录格式。
